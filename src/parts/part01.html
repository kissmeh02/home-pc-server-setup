architecture notes
Ryzen 9 5950X is the highest-end AM4 CPU available — 32 threads gives each of the 7 VMs a dedicated allocation with 12 threads left for the Proxmox host, ZFS ARC, and overhead. The step up from 5900X (24t) is meaningful at this VM density.
X570 Dark Hero chosen specifically for Proxmox IOMMU: each PCIe x16 slot is in its own IOMMU group on this board. Verified by the Proxmox community. Critical for dual GPU passthrough.
The Mellanox ConnectX-4 Lx gives you 25G connectivity now, upgradeable to 100G later by swapping to ConnectX-6 Dx when you add a 100G switch. Same SFP28 DAC cable works at both speeds.
ZFS on the 2× 20 TB IronWolf Pro: set ARC size to 16 GB in Proxmox to keep ZFS metadata hot. The 990 Pro 4 TB NVMe can also serve as an L2ARC cache for the HDD pool if VM I/O patterns warrant it.
VFIO GPU passthrough: both 4070s are identical. Bind each by PCI bus address in /etc/modprobe.d/vfio.conf — not by vendor:device ID alone. Use `lspci -nn` to get addresses before writing the config.
